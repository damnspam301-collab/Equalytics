Equalytics: Gender Pay Gap Analyzer

Project by Greenfield Chennai International School - Class XII (2025-2026)

Introduction

The gender pay gap represents persistent differences in salaries between men and women. This project uses AI to analyze data and uncover biases in pay structures.

Problem Statement

Despite progress, gender inequality affects employment and wages. Our focus is on the gender pay gap, where women earn less for similar work.

Our Solution

We built a linear regression model to predict salaries and detect disparities. Use the tool below to analyze sample inputs.
